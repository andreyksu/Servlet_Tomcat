Подготовка СУБД и деплой ДБ	(в качестве СУБД используется Postgresql)
	1. Установите Postgresql исходя из используемой ОС.
	2. Добавьте пользователя testov:
		createuser -PE -l -d -e -l testov
	3. Создайте базу данных:
		create database testdb WITH OWNER = testov ENCODING = 'UTF8';
	4. Выполните скрипт:
		psql -U testov -d testdb -h <host_name> -f <script>;

Подготовка серверной составляющей (в качестве сервера используется контейнер сервлетов Tomcat):
	1. Установите Open-JDK исходя из используетмой ОС.
	2. Установите Tomcat исходя из используемой ОС.
	3. Установите maven исходя из используемой ОС.
	4. Соберите продукт командой:
		mvn clean package
	5. Задайте параметры подключения к БД в файле:
		./webapp/src/main/webapp/META-INF/context.xml
	6. Задеплойте приложение в Tomcat согласно инструкции Tomcat

Использование:
	1. Запустите браузер введя адрес машины и url чата
		http://<hostname>:8090/ChatOnServlet/
	2. Для регистрации перейдите на форму регистрации
	3. После регистрации можно отправлять сообщения.
